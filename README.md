Slim 4 skeleton with basic parallax homepages
## install
clone to your htdocs
```sh
git clone <link>
```
install depencies
```sh
composer install --no-dev
yarn install
```
go to `htdocs/app/container.php`

comment this
```php
// $app->setBasepath();
```
